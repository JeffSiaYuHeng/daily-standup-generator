# Daily Standup AI - Tech Stack

## Core Frontend
- **React 19**: Modern UI framework with Concurrent Mode.
- **Vite**: Ultra-fast build tool and dev server.
- **TypeScript**: Static typing for reliability.

## Styling & UI
- **Tailwind CSS**: Utility-first CSS framework.
- **Transparent Design**: Custom glassmorphism implementation using `backdrop-blur`.
- **Heroicons**: High-quality SVG icons for UI.
- **Dark Mode**: Native system-aware theme switching.

## AI Engine
- **Google Generative AI SDK**: Direct integration with Gemini models.
- **Gemini 1.5/2.0**: Used for natural language processing and structured text generation.

## Backend & Persistence
- **Supabase**: Backend-as-a-Service providing PostgreSQL and Authentication.
- **localStorage**: primary local persistence for offline-first capability.
- **Web Speech API**: Browser-native voice-to-text integration.

## PWA Features
- **Vite PWA Plugin**: Handles service worker generation.
- **Manifest**: Custom icons and splash screens.
- **Offline Support**: Cache-first strategy for core assets.

## Dev Tools
- **ESLint**: Code quality.
- **PostCSS**: CSS processing.
- **UUID**: Robust client-side ID generation.
